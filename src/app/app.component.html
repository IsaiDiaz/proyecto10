Introducción a la asignatura

En la materia de "tecnologias web" se aprende sobre las tecnologias de internet, tecnologias web para la presentacion, programacion en el servidor, programacion en el cliente, arquitectura orientada a servicios y frameworks.

Concepto de Framework

Un framework es una estructura que se puede aprovechar para desarrollar un proyecto. El framework es como una plantilla, un esquema conceptual, que simplifica la elaboracion de una tarea, ya que solo es necesario complementarlo de acuerdo a lo que se quiere realizar

Principales framworks del 2021

  1. React

    Es un framework altamente escalable basado en JavaScript y capaz de satifacer una demanda masiva. Algunos sitios que utilizaron React son: Netflix, Facebook, Airbnb y Reddit.

  2. Angular

    Es un framework ideal para la creacion de aplicaciones de una sola página y faciles de mantener basado en JavaScript y TypeScript .Weather, PayPal, Netflix y Youtube de PS3 utilizan Angular.

  3. Django

    Este framework utiliza el lenguaje de Python que permite una codificacion limpia y eficiente. Reconocido por su excelente seguridad incluida y uso altamente intuitivo, versatil y facilmente escalable. Pinterest, Instagram y Quora usan Django.

  4. Ruby on Rails

    Ruby on Rails utliza en modelo MVC. Utiliza el lenguaje Ruby que permite el desarrollo de aplicaciones en periodos de tiempo mas cortos. Shopify, Airbnb y Groupoon usan Ruby on Rails.

  5. Laravel

    Ideal para principiantes, utiliza el lenguaje PHP y se basa en el modelo MVC. Lo malo es que no soporta proyectos grandes y usuarios avanzados lo pueden encontrar limitante.

  6. ASP.net

    Desarrollado por Microsoft, ASP.NET utiliza el lenguaje C# y se usa pra crear aplicaciones web para PC y dispositivos móviles. Getty Images, Taco Bell y Stack Overflow utilizaron ASP.NET.

  7. Express.js

    Es uno de los frameworks mas flexibles, su sistema de complementos, la compatibilidad con otros frameworks y otras caracteristicas lo hacen uno de los frameworks candidatos al mas usado en los proximos años. Utiliza el lenguaje JavaScript u lo utilizan Storify, IBM y Uber.

  8. Vue.js

    Fue pensado principalmente como una alternativa más ágil para los usuarios de Angular. Este framework se puede adoptar en porciones aisladas de proyectos sin ningun problema. Es facil de utilizar para principiantes y suficientemente versatil para usuarios avanzados y utiliza el lenguaje JavaScript.

  9. Spring

    Es el framework MVC más popular para el lenguaje Java, es utilizado por grandes entidades como Wix y Ticketmaster. Los expertos en Java pueden utilizar Spring para desarrollar aplicaciones de alto nivel diseñadas para manejar trafico masivo.

  10. Ember

    Este framework es el favorito para los desarrolladores que valoran el desarrollo de aplicaciones robustas y escalables. Es altamente productivo desde el primer momento y se esta utilizando para aplicaciones como Apple Music, LinkedIn, Netflix y Google.

Conclusiones

Más alla de cual sea el mejor framework actualmente, es importante mantenerse actualizado y capacitado para utilizar diferentes frameworks según sea conveniente. Por lo que, el constante apredizaje y experimentacion son esenciales para el desarrollo profesional.
